What is Vue.js?

  • Vue.js is MVVM framework like React.js or Angular.

MVVM stands for Model View View-Model which in practice means that framework's main concern is to maintain consistency between Model (Javascript object) and View (DOM presentation).

In v2 Vue.js implement VirtualDOM pattern to do it's magic. It means that all DOM updates are first done in memory and then 'diffed' with current DOM state, afte that only changes are propagated into DOM.

Why I love Vue.js

  • It really made me enjoy implementing business logic on frontend (think: Baskets with dynamic calculations, discounts etc.)
  • Component based approach
  • Data driven real-time dashboards is a breeze because of watchers and computed properties
  • Data driven DOM
  • Paradigm shift

Why I choose Vue.js over React.js/Angular/Ember...?

  • As Laravel's official frontend framework Vue.js gained a lot of exposure
  • Lightweight - you can use it to simply spice up HTML or use it as full-blown application framework
  • Personal preference. After skimming through Angular and React.js docs Vue.js simply clicked for me.
  • Low entry point barrier. Simple, easy and elegant.
